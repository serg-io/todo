This web application is an implementation of the [TodoMVC](http://todomvc.com/) project. It is
built using [template-infuse](https://github.com/serg-io/template-infuse), custom elements, ES2015,
and [webpack](https://webpack.js.org/).

This application is available at: [todo.alcantara.io](http://todo.alcantara.io/)

### Development ###

To run this application in your localhost, clone this repository, execute `npm install`, to install
all dependencies, followed by `npm start`, to start the webpack development server, and go to
**http://localhost:8080/** in your browser.
